Layout

Sidebar (sticky): Logo “SmartChef OS”, links: Dashboard, Order Line (default), Manage Tables (disabled), Manage Dishes, Customers, Settings, Help, Logout. Active pill with icon+label.

Topbar: Search input, user chip (avatar + role), quick actions (Print, KOT) as ghost buttons.

Page: Order Line (default)

Status chips row: All / Dine-in / Waitlist / Takeaway / Served (with counts). Use Chips.tsx.

Active order carousel: small cards showing #FOO27, table, type, “In Kitchen / Ready”.

Category strip: pills (All, Specials, Soups, Desserts, Chicken, etc.).

Menu grid: ItemCard with image, title, price, + / – qty, availability dot; grid/list toggle; FilterDrawer (veg, price range, popularity).

Right CartPanel (sticky): table no., guests, line items with qty editor & modifiers; TotalsCard (subtotal, tax, service charge, discount input); PaymentButtons (Cash / Card / Scan); CTAs: Send to Kitchen (primary), Print Bill, Place Order (show toasts only).

Keyboard shortcuts: F1=Search, F2=Add item (focus first add), F3=KOT, Ctrl+P=Print Bill. Include a small ShortcutsModal.

Page: Manage Dishes

Left sub-sidebar: categories with counts; “Add New Category” button (client-side only).

Top controls: search, Add New Dish (opens AddDishModal with fields: name, category, price, photo upload, veg toggle, tax slab, printer route).

Grid of dish cards with availability toggle, edit/delete checkbox; include dashed “Add New Dish” card.

Page: Customers

Table with columns: Name, Phone, Last Visit, LTV, Favorites, action Create Order.

Simple row filter/search.

Page: Dashboard (read-only mock)

Greeting panel: “Hello Charan, ready to thrive today?”

StatCard trio: Daily Sales, Orders, Avg Ticket Size.

InsightCard: “Sales up 12% this week. Best-selling: Paneer Butter Masala.”

Charts: lightweight line + donut (use a tiny lib like recharts or simple SVG placeholders). Keep colors in brand palette.

Interaction & UX Rules

Big, accessible tap targets (≥44px), visible focus states, ARIA labels on CTAs.

Rounded corners (16–24px), soft shadows, calm animations (hover raise, button press).

Toasts: “Item added”, “KOT sent”, “Bill printed”.

Persistent cart with localStorage.

Theme & Tokens (Tailwind classes)

Buttons:

Primary: bg-brand-primary text-white rounded-xl px-5 py-3 shadow-sm hover:opacity-95

Secondary: bg-card text-brand-ink border border-brand-primary/20

Ghost: text-brand-ink/70 hover:bg-card

Cards: bg-card rounded-2xl shadow-md p-4

What NOT to implement

No backend/auth, no real printing, no hardware integration.

No heavy state libs—use React context or simple hooks.

No complex analytics—static mock text is fine.
